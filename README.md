# LifeSense

每日生活状态监测助手，帮你量化记录心情、睡眠、精力等维度，自动计算 Life Score 并给出建议。

## 功能

- **每日打卡** — 记录心情、精力、压力、专注、睡眠、饮水、活动和备注
- **Life Score** — 基于 PHQ-9 / WHO-5 / DASS-21 等循证量表加权评分，生成状态标签与分步骤可执行建议
- **历史记录** — 浏览、筛选（分数段）、编辑、删除单条记录，编辑后展示前后分数对比
- **周报趋势** — 按 ISO 周汇总各维度均值，多周横向对比
- **7 天趋势图** — 可视化近一周各维度变化，连续打卡天数统计
- **Windows 桌面版** — 安装后从开始菜单或桌面快捷方式直接启动，本地保存记录、头像、兴趣和周目标
- **离线同步队列** — 离线时新增、编辑和删除先保存到本机，恢复网络后可重试同步
- **本周目标** — 在首页设置睡眠、饮水、专注目标，并在最新周报查看进度与下周行动建议
- **每日速览** — 所有新闻兴趣共享实时 RSS 源池，按兴趣筛选；Windows 与移动端使用统一新闻卡片排版和详情体验
- **每日提醒** — 支持本地提醒、时区适配及排程失败后的重试
- **深色模式** — 跟随系统自动切换
- **访客模式** — 无需账号，数据存本机，即装即用
- **云同步** — 注册账号后自动同步，多端一致，卸载重装可恢复
- **头像 & 昵称** — 个人资料云同步

## 技术栈

Flutter 3 · Dart · 华为 AGConnect (Auth / CloudDB / Cloud Storage) · Material 3

## 普通用户：直接下载安装

前往 [Releases](https://github.com/myzane678/LifeSense/releases) 下载适合设备的安装包。

### Windows 电脑

下载 `LifeSense-1.5.0-Setup.exe` 并双击安装。安装后可从开始菜单启动，也可在安装向导中选择创建桌面快捷方式。

> 安装包当前未进行代码签名，Windows 首次运行可能显示“未知发布者”提示；确认来源为本项目 Release 后即可继续安装。

### Android 手机

下载 `LifeSense-1.5.0.apk`，安装后即可使用。

1. 在手机浏览器打开本页面，点击 APK 文件下载；或在电脑下载后通过数据线 / 微信 / QQ 传到手机
2. 在手机文件管理器找到下载的 APK 文件，点击安装
3. 若出现「安装未知应用」或「来源未知」提示，点击「仍然安装」或在设置中允许该来源即可，这是 Android 对非应用商店安装包的正常提示
4. 安装完成后打开 LifeSense，可选择注册账号或以访客身份直接使用

- **无需任何配置**，安装完直接打开
- 支持**访客模式**：不注册账号，数据仅存本机，也能完整使用所有记录功能
- 注册邮箱账号后，数据自动云同步，换机或重装后可恢复

## 开发者：本地构建

本项目云同步基于华为 AGConnect。自行构建时需完成以下步骤：

1. 在 [AGConnect 控制台](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html) 创建 Android 应用（包名 `com.example.life_sense`）
2. 开通 Auth、CloudDB、Cloud Storage 三项服务
3. 下载 `agconnect-services.json`，放至以下两处：
   - `android/app/agconnect-services.json`
   - `android/app/src/main/assets/agconnect-services.json`
4. 在 CloudDB 创建 zone `lifeSense`，导入 object type（见 `packages/agconnect_clouddb/android/.../objecttypes/`）

```bash
flutter pub get
flutter run
flutter build apk --release
```

## Changelog

### v1.5.0 (2026-07-13)

- **Windows 桌面版**：新增 Windows 本地模式与安装程序，可从开始菜单或桌面快捷方式直接启动；Windows 下记录、头像、兴趣和周目标仅保存在本机，暂不支持登录、云同步、拍照和后台每日提醒
- **离线同步可靠性**：记录新增、编辑、删除均先保存本机；同步失败进入按账号隔离的持久化队列，恢复网络后可重试；删除使用 tombstone 防止旧云端记录复活
- **每日提醒可靠性**：使用设备时区安排本地提醒；排程和云同步失败会保留待处理状态并支持重试
- **本周目标**：首页新增睡眠、饮水、专注目标入口，最新周报展示目标进度和下周行动建议
- **每日速览**：所有新闻兴趣共用实时 RSS 源池并按规则筛选；新闻卡片统一固定摘要区，Windows 与移动端保持一致排版；Windows 新闻详情使用可滚动对话框
- **测试与发布**：新增离线同步、周目标和 Windows 新闻卡回归测试；提供 Windows 安装包与 Android APK
